# financial_databases_guide
![Uploading d9962e54f21df287f5679d9eb6b40b82.jpg…]()

本数据库系统性整理了国内外主流的金融数据源，涵盖免费与付费两大类别，适用于从个人学习到专业机构研究的全场景需求。免费数据库部分包含Tushare、AKShare、Baostock、Yahoo Finance等，覆盖A股、港股、美股、期货、宏观经济等核心数据，具备零门槛接入、Python生态完善等优势，适合入门与非实盘量化研究；付费数据库则涵盖Wind、同花顺iFinD、Bloomberg、MSCI Barra等机构级平台，提供实时行情、深度财务数据、量化因子与另类数据（如舆情、卫星图像），满足专业量化策略开发与跨境投资研究需求。文件还附有选型指南，依据“A股入门”“量化回测”“海外研究”“机构级分析”等场景推荐最优工具，并提示调用限制、数据延迟与字段命名差异等关键注意事项，是高效选型与开发的实用参考手册
